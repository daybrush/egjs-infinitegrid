InfiniteGrid v2.1.1-snapshot

A module used to arrange card elements including content infinitely on a grid layout. With this module, you can implement a grid-pattern user interface composed of different card elements whose sizes vary. It guarantees performance by maintaining the number of DOMs the module is handling under any circumstance

Getting Started

Browser support

IE 8+ (possibly 9 also), latest of Chrome/FF/Safari, iOS 7+ and Android 2.1+ (except 3.x)

Quick steps to use:

Set up your HTML

<ul id="grid">
    <li class="card">
        <div>test1</div>
    </li>
    <li class="card">
        <div>test2</div>
    </li>
    <li class="card">
        <div>test3</div>
    </li>
    ....
</ul>

Load files or import library

ES5
<script src="//naver.github.io/egjs-infinitegrid/https://naver.github.io/egjs-infinitegrid/v3.0.0/dist/infinitegrid.pkgd.js"></script>
ES6+
import InfiniteGrid from "@egjs/infinitegrid";

Initialize

// create InfiniteGrid ui without option
new eg.InfiniteGrid("#grid");

// create InfiniteGrid ui with options
new eg.InfiniteGrid("#grid",{
  itemSelector: ".item"
});

Initialize Layout

// create InfiniteGrid
const ig = new eg.InfiniteGrid("#grid");

// initialize layout
// GridLayout(default), JustifiedLayout, FrameLayout, SquareLayout, PackingLayout
ig.setLayout(eg.InfiniteGrid.JustifiedLayout, {
    itemSize: 200,
});
